# Frans Kevin Portfolio Website

Website portofolio futuristik minimalis berbasis HTML, CSS, dan JavaScript.

## File utama

- `index.html` — struktur halaman portofolio.
- `style.css` — desain visual futuristik minimalis.
- `app.js` — animasi, navigasi, reveal effect, dan integrasi GitHub API.

## Cara menghubungkan ke GitHub

1. Buka file `app.js`.
2. Cari bagian:

```js
const GITHUB_USERNAME = "franskvs";
```

3. Ganti `franskvs` dengan username GitHub kamu.
4. Simpan, lalu jalankan `index.html` di browser.

## Deploy ke GitHub Pages

1. Upload semua file ke repository GitHub.
2. Buka repository → Settings → Pages.
3. Pilih branch `main` dan folder `/root`.
4. Klik Save.
5. Website akan aktif melalui link GitHub Pages.
